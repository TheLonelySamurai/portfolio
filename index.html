Samurai Showdown *

*name not finalized

A 2-player Platform Fighter

Samurai Showdown is one of my first projects that I have created to completion. The project has one mode and two characters, one for player one and one for player two. The game features 1 on 1 combat using arc shapes as sword swings and rect shapes as a projectile with jumps, a double jump, a glide on player two, and different movement stats.

Play the game!

Graphing Game Statistics

Using Code to Graph a CSV file

One of my other projects was to use a CSV file containing statistics in a sheet format and convert it into a graph for visualization. This is a very successful adaptation of this containing a CSV from wikipedia data of all the top 256 games from 2016 with different formats you can look at by using the arrow keys.
